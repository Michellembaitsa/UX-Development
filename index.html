CSS Box Model.

What is the box model and what properties does it have?

The box model Box model refers to the design and layout of a webpage. It wraps around every HTML element. It consists of the following properties:

  1. Content-This refers to the content of the box, where the texts and images appear.
  2. Padding-This is the area surrounding the content and is transparent in color.
  3. Border-This is what goes around the padding and the content
  4. Margin-This clears an area around the border and is also transparent.

Calculating a box model.

Given the width of a box model as 400px, padding as 20px, margin as 20px and border as 2px, find the total width of a paragraph.

Paragraph width=400px-(2px+20px+20px) Thus, the width of the paragraph(content)will be 358px.
